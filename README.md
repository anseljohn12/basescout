# BaseScout 🚀

Discover trending pools and hidden gems on Base.

BaseScout is a lightweight analytics and discovery platform built for the Base ecosystem. It helps users identify promising liquidity pools by analyzing volume, liquidity, buy/sell activity, and market momentum.

## Features

* 🔥 Hot Pool Detection
* 💎 Gem Pool Discovery
* 📊 Volume Analytics
* 💧 Liquidity Tracking
* 📈 Buy/Sell Activity Monitoring
* ⚡ Base Ecosystem Focus

## Live Application

https://basescout-ten.vercel.app

## Tech Stack

* Next.js
* TypeScript
* DexScreener API
* GeckoTerminal API
* Vercel

## Why BaseScout?

The Base ecosystem is growing rapidly. BaseScout helps users discover opportunities faster by surfacing pools with strong activity and potential.
